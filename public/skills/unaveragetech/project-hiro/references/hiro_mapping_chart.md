## How It Works

1. Character is converted to 8-bit binary
2. Binary is padded to multiple of 3 bits
3. Binary is split into 3-bit groups
4. Each 3-bit group is mapped to a hieroglyphic symbol
5. Symbols are concatenated to form the encoded representation

Note: Hiro uses additional encryption (XOR with key stream) before this visual encoding.
