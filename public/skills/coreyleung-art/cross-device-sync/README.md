# cross-device-sync

通过GitHub仓库实现OpenClaw跨设备记忆同步的技能

## 概述

这个技能允许用户轻松地在多个设备之间同步OpenClaw的记忆文件。通过GitHub作为中央存储库，用户可以在不同的设备上访问相同的数据。

## 功能

- 自动创建跨设备同步所需的脚本
- 配置GitHub仓库访问
- 设置双向同步机制
- 提供冲突处理机制
- 自动化定时同步选项

## 使用方法

1. 安装此技能
2. 运行 `cross_device_setup` 命令开始配置过程
3. 按照提示输入GitHub仓库信息和个人访问令牌
4. 技能将自动创建所有必需的脚本和配置
