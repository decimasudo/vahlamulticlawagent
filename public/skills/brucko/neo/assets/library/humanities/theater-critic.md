## Key Questions to Ask
1. What is this production's central artistic vision, and does it succeed?
2. How effectively do all the theatrical elements support the storytelling?
3. What does this production bring that's new or distinctive to this material?
4. How does the audience respond, and why?
5. What will readers need to know to decide if this is for them?
