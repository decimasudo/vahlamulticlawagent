# 港股每日热门股票追踪

## 任务说明
获取港股当日热门股票信息，包括涨幅榜、成交量榜、沪港通成交额，并提供简要分析。

## 数据来源
- 东方财富港股行情
- 同花顺港股数据
- 新浪财经港股

## 输出格式
```
========== 港股每日简报-{日期} ==========

📊 今日市场概况
- 恒生指数: 点位 / 涨幅%
- 恒生科技指数: 点位 / 涨幅%
- 成交额: XXX亿

🔥 涨幅榜 Top5
1. [股票代码] [股票名] (+X.XX%)
   - 原因: [板块/概念/消息]

2. [股票代码] [股票名] (+X.XX%)
   - 原因: ...

3. ...

📊 成交活跃榜 Top5
1. [股票代码] [股票名] (成交额: XX亿)
   - 原因: ...

2. ...

🔄 沪港通(南向)成交 Top3
1. [股票名] (XX.XX亿元)
2. ...
3. ...

💡 热点板块
- [板块名]: [领涨/领跌]，相关股票: XXX

=============================
```

## 热门原因分类
1. **业绩驱动** - 财报超预期/预增
2. **概念炒作** - AI/新能源/半导体等
3. **回购支撑** - 大额股票回购
4. **资金流入** - 沪港通大幅买入
5. **消息刺激** - 重大合作/产品发布
6. **板块轮动** - 行业轮动补涨

## 响应要求
1. 所有信息用中文
2. 股票代码使用港股标准格式 (如 0700.HK)
3. 成交额单位统一为"亿元港币"
4. 原因分析简洁明了，不超过2句话
5. 如当日数据不可用，注明"数据获取中"
