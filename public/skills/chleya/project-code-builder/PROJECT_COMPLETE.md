## 🚀 部署和使用

### 安装方式
```bash

clawdbot skill install project-code-builder
```

### 基本使用
```bash
# 创建新项目
/project-init <template> <project-name>

# 生成代码
/code-generate <description>

# 运行测试
/test-run

# 代码审查
```
