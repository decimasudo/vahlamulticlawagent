# Security Joes AI Analyst

SecOps skill for endpoints: EDR, Sysmon, EVTX on heartbeat, least privilege, network visibility, credential protection, device inventory and known vulnerabilities, weekly assessment, and skill integrity.

## Publish to ClawHub

The [ClawHub upload form](https://www.clawhub.ai/upload) does **not** read slug/display name from files — you must **type them in**:

| Field         | Value                      |
|--------------|----------------------------|
| **Slug**     | `security-joes-ai-analyst` |
| **Display name** | `Security Joes AI Analyst` |
| Version      | `1.0.0` (default)          |
| Tags         | `latest` (or add: security, secops, edr, sysmon, evtx) |
| Changelog    | Optional; describe changes for this release. |

Then **drop this folder** (or a zip of it), add changelog if needed, and click **Publish skill**.

## Files

- **SKILL.md** – Skill content (instructions, responsibilities, templates).
- **skill.json** – Metadata (slug, displayName, summary, version) for reference; ClawHub upload form does not auto-fill from it.
- **README.md** – This file.
